# Cryptography Web Game

## Overview
This project is a web-based application designed to teach and demonstrate classical cryptographic techniques through interactie challenges.  

## Features
- Multiple cipher-based challenges
- Interactive gameplay and user progression
- Educational explanations for each cipher 

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js

## Setup and Run
### Prerequisites
- Node.js installed

### Installation
- npm install

### Running the Project
- node server.js (in integrated terminal for server.js file)
- drag home.html into a browser
