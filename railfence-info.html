Rail Fence Cipher

The Rail Fence Cipher is another transposition cipher. It simply writes the plaintext in a zigzag pattern across a set number of rows then reads it row by row to scamble the text.

The Rail Fence Cipher can be easily cracked through brute force.

For example: Plaintext → Curiouser and curiouser → CURIOUSERANDCURIOUSER
